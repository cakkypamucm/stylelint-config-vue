This is `vue2+` `stylelint` config (`scss` + `prettier`)

### Install

```shell
npm i --save-dev github:cakkypamucm/stylelint-config-vue
```

### Use

Add `stylelint-config-vue-cakkypamucm` to the "extends" array in your `stylelint` config file

<pre>
{
    "extends": ["some-other-config-you-use", <b>"stylelint-config-vue-cakkypamucm"</b>]
}
</pre>

### For synergy

Add [eslint-config-vue-cakkypamucm](https://github.com/cakkypamucm/eslint-config-vue) to your project
